Project Description

The Food Delivery App is a user-friendly platform designed for ordering food from local restaurants with seamless navigation and a simple checkout process. The app allows users to browse various menus, select their meals, and track the delivery in real-time. It provides an intuitive interface for both customers and restaurant owners, integrating payment gateways for easy and secure transactions.

Technology Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask)
  • Database: MySQL
  • Payment Integration: Stripe API
  • Deployment: Heroku
